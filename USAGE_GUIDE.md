# CWJ TOOLS — 使用指南

> 写给完全不懂技术的朋友。

---

## 这个项目是什么？

CWJ TOOLS 是一个 **AI 工具平台的网站**，就像 ChatGPT、文心一言那样的 AI 对话网站，但功能更多。

它包含 **8 个页面**：

| 页面 | 干什么用的 |
|------|-----------|
| 首页 (index.html) | 展示平台介绍、工具列表、数据统计 |
| 登录页 (login.html) | 登录和注册账号 |
| 控制台 (dashboard.html) | 查看使用数据、图表、最近对话 |
| AI 聊天 (chat.html) | 和 AI 对话，问问题、写代码 |
| Agent 中心 (agent.html) | 管理 AI 智能代理 |
| 模型管理 (models.html) | 选择不同的 AI 模型 |
| 设置中心 (settings.html) | 修改主题、通知等偏好 |
| 用户中心 (profile.html) | 查看个人信息、用量、成就 |

---

## 目前是什么状态？

**这是一个纯前端演示版。**

什么意思呢？就是说：
- ✅ 页面打开就能用，不需要安装任何东西
- ✅ 你登录、聊天、改设置，数据会保存在你的浏览器里（叫 localStorage）
- ✅ 刷新页面，数据还在
- ❌ 换一台电脑/手机，数据就没了（因为数据存在你自己的浏览器里，没有存在服务器上）
- ❌ 清除浏览器缓存，数据也会丢失

**简单理解：** 就像一个记事本，数据存在你自己的手机上，不会上传到云端。

## 怎么在自己电脑上打开看效果？

### 方法一：Python（最简单）

1. 把项目文件夹放到电脑上
2. 打开终端/命令行，进入项目文件夹
3. 输入：
   ```
   python3 -m http.server 8080
   ```
4. 打开浏览器，访问 `http://localhost:8080`

### 方法二：VS Code

1. 安装 VS Code 编辑器
2. 安装 "Live Server" 插件
3. 右键点击 `index.html`，选择 "Open with Live Server"

### 方法三：直接双击

直接双击 `index.html` 文件，浏览器就能打开（但地址栏会显示本地路径，不是正常的网址）

## 以后想接真实后端怎么办？

目前所有数据存在浏览器本地。如果想让数据存在服务器上（比如多台设备同步、数据不会丢失），需要：

1. **看 API_SPEC.md** — 里面列出了所有需要的接口（比如"登录"、"发消息"、"获取数据"），每个接口都写清楚了要传什么参数、返回什么数据
2. **看 BACKEND_GUIDE.md** — 里面说明了用 Node.js 怎么搭建后端服务器，需要哪些数据库表
3. **改 api.js** — 这个文件是"数据中转站"，目前内部用 localStorage 实现。以后把内部实现改成调用服务器接口就行，其他页面代码完全不用动

| 文件 | 干什么的 |
|------|---------|
| `index.html` | 首页 |

## 特别功能说明

- **深色/浅色切换**：点右上角的太阳/月亮图标
- **手机适配**：用手机浏览器打开也能正常使用
- **AI 聊天**：目前是模拟回复（不是真的 AI），以后接上后端就是真的
- **数据导出**：控制台和聊天页有"导出"按钮，可以下载 CSV 文件
