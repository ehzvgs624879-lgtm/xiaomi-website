def quicksort(arr):
    """
    快速排序算法
    时间复杂度: O(n log n) 平均, O(n²) 最坏
    空间复杂度: O(log n)
    # 基线情况：数组长度 <= 1 时直接返回
    if len(arr) <= 1:
        return arr
    
    # 选择基准元素（这里选中间元素）
    pivot = arr[len(arr) // 2]
    # 三路划分
    left = [x for x in arr if x < pivot]   # 小于基准
    mid = [x for x in arr if x == pivot]    # 等于基准
    right = [x for x in arr if x > pivot]   # 大于基准
    # 递归排序并合并
    return quicksort(left) + mid + quicksort(right)
核心思想：选取一个基准值（pivot），将数组分为三部分，递归排序。

优化建议：
• 使用 random.choice(arr) 随机选基准，避免最坏情况
• 对小数组（n < 10）切换为插入排序
• 使用原地分区（Lomuto/Hoare partition）减少内存分配

归并排序实现：

def merge_sort(arr):
    """归并排序 - 稳定的分治排序算法"""
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)
def merge(left, right):
    """合并两个有序数组"""
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
对比：
指标快速排序归并排序
时间(平均)O(n log n)O(n log n)
空间O(log n)O(n)
